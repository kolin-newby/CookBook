# Cookbook App

This project is a relatively simple web application that is built using React and JavaScript on the frontend and leverages [Supabase](https://supabase.com/) as a backend. To be more specific, this app implements user authentication, data queries, and data mutation through the Supabase integration and the database therein.

## Frontend Tech Stack
- JavaScript
- React.js
- [TanStack Queries](https://tanstack.com/query/latest)
- [Lucide](https://lucide.dev/) Icon Library
